

Moviex Web Site: 

	

 	This web-based platform allows users to register, login, and logout.
	They can view the list of available movies and purchase tickets for their preferred movie. On the other hand,
	the admin can access and manage all the orders made by users. Additionally, the admin can edit, add, 
	or delete information about movies, actors, producers, and cinemas. This system ensures a seamless and user-friendly
	experience for both users and admin

Moviex Technology :

	 Asp.Net Web Api | Angular |MySQL.
